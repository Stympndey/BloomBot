To make BloomBot work correctly, you need to set up your environment variables (usually in a .env file or your hosting provider's dashboard):
Variable	Description
SESSION_ID	Your unique WhatsApp session ID.
PREFIX	The character used to trigger commands (e.g., . or !).
OWNER_NUMBER	Your WhatsApp number (with country code).
SUDO	Additional numbers that can access admin commands.
MODE	Set to public or private.
